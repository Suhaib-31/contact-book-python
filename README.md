# Contact Book Application 📞

## Overview

A Python-based Contact Book application that allows users to store, search, view, and delete contacts using a simple command-line interface.

All contact information is saved in a local text file for persistent storage.

---

## Features

- Add new contacts
- View all contacts
- Search contacts by name
- Delete contacts
- File-based data storage

---

## Technologies Used

- Python 3
- File Handling
- Functions
- CRUD Operations

---

## How to Run

```bash
python main.py
```

---

## Sample Menu

```text
1. Add Contact
2. View Contacts
3. Search Contact
4. Delete Contact
5. Exit
```
